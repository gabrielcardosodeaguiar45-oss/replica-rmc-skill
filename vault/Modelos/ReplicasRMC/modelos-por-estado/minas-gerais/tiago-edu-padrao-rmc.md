# Modelo Tiago+Edu × padrão × MG (RMC)

Versão genérica MG para bancos diversos. Mantém tratamento robusto da decadência (desafio local).

## Quando usar

1. Banco réu é Bradesco, Itaú, Daycoval, Safra, Parati, C6, Mercantil (embora Mercantil tenha cenário próprio — ver [[tiago-edu-sem-contrato-mg|Sem contrato]]).
2. Não há particularidade de BMG/PAN.
